# World Boss Group Helper

WoW Classic addon for managing world boss PUG recruitment.

## Install

Copy to `World of Warcraft/_classic_/Interface/AddOns/`

## Usage

Start recruiting:
```
/wbgh azuregos
```

With hard reserve:
```
/wbgh azuregos Mature Blue Dragon Sinew
```

Stop recruiting:
```
/wbgh off
```

Click the on-screen buttons to spam Trade (in cities) or LFG channels.

## Settings

```
/wbgh set raidsize 40
/wbgh set tank azuregos 2
/wbgh set healer kazzak 8
/wbgh set hr azuregos Mature Blue Dragon Sinew
/wbgh set tradeinterval 300
/wbgh set lfginterval 300
```

Default bosses: Azuregos, Kazzak, Emeriss, Lethon, Taerar, Ysondre
